# Hackathon Starter 2024

> A boilerplate project for utilizing my current prototyping preferences for 2024

Here is a project that configures the following tools in a monorepo:

- UI
  - React with TypeScript
  - React Query
  - TailwindCSS
  - ShadCN
- API
  - Express.js

# Getting Started

```bash
npm install
npm run dev
```
